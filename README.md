# Mini Notes App (PHP + MySQLi)

Mini Notes App is a simple CRUD application for creating and managing notes, built with plain PHP using the **MySQLi** extension.

This project was created for learning purposes to practice working with databases, prepared statements, and basic PHP application structure.

---

## Features

- Create notes
- View notes list
- Edit notes
- Delete notes
- Note status: `draft` / `published`
- SQL Injection protection (prepared statements)
- Basic XSS protection when displaying data

---

## Technologies Used

- PHP (procedural style)
- MySQL
- MySQLi
- HTML
- Git / GitHub
